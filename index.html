Hallo !

Hier möchte ich zeigen wie man einen Conrad RSL Switch mit dem Photon Particle steuert.

Ein Conrad RSL Switch ist eine per Funk gesteuerte Steckdose. (siehe www.conrad.de)

Voraussetzungen

Dieser Artikel setzt voraus, dass Du folgendes beherrscht:

PARTICLE APP auf Deinem Smartphone installieren Mit TINKER die blaue LED and D7 ein- und ausschalten

BLYNK APO auf Deinem Smartphone installieren Mit der WEB IDE die blaue LED and D7 ein- und ausschalten.

BLYNC WEB IDE

In der IDE die Library RSL_SWITCH suchen.

Dann "USE THIS EXAMPLE AUSWÄHLEN"

Ergebnis : Ein neue APP mit dem Namen conrad-rsl-example.ino wird erstellt.

IDE Starten : LINK
